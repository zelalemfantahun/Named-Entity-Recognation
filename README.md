# 🏷 Named Entity Recognition (NER)

A Natural Language Processing project that automatically identifies and classifies named entities such as people, organizations, locations, dates, and other important information from unstructured text.

---

## 📌 Features

- Named Entity Recognition (NER)
- Text Preprocessing
- Tokenization
- Feature Extraction
- Model Training & Evaluation
- Entity Classification

---

## 🚀 Technology Stack

- Python
- NLTK
- spaCy
- Pandas
- NumPy
- Scikit-learn
